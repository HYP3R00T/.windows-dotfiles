# .windows-dotfiles

## PowerShell

### Prerequisites

- Install [`winfetch`](https://github.com/lptstr/winfetch)

```sh
Install-Script winfetch
```

- Import-Module [`PSReadLine`](https://github.com/PowerShell/PSReadLine)

### Configuration

Move the entire `PowerShell` folder to `%USERPROFILE%\Documents`

To add more awesomeness check out [awesome-powershell](https://github.com/janikvonrotz/awesome-powershell).
