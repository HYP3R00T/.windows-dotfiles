# .windows-dotfiles

This repo is a collection of my personal dotfiles of windows. 

## Install

- [PowerShell](https://learn.microsoft.com/en-us/powershell/scripting/install/installing-powershell-on-windows)
- [starship.rs](https://starship.rs/)
- [winfetch](https://github.com/lptstr/winfetch)
- [PSReadLine](https://github.com/PowerShell/PSReadLine)

## Setup

- Clone this repo inside `%USERPROFILE%`. Open PowerShell
```sh
cd $HOME
git clone https://github.com/HYP3R00T/.windows-dotfiles
explorer .
```

- Move the following folders to mentioned locations. 
    - `PowerShell` -> `%USERPROFILE%\Documents`
    - `.config` -> `%USERPROFILE%`

## Extra

To add more awesomeness check out [awesome-powershell](https://github.com/janikvonrotz/awesome-powershell)
