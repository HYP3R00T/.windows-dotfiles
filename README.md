# .windows-dotfiles

This repo is a collection of my personal dotfiles of windows. 

## Install

- [starship.rs](https://starship.rs/)
- [`winfetch`](https://github.com/lptstr/winfetch)
- [`PSReadLine`](https://github.com/PowerShell/PSReadLine)

## Setup

- Clone this repo inside `%USERPROFILE%`
```sh
cd $HOME
git clone https://github.com/HYP3R00T/.windows-dotfiles
explorer .
```

- Move the following folders to mentioned locations. 
    - `PowerShell` -> `%USERPROFILE%\Documents`
    - `.config` -> `%USERPROFILE%`
